  1. Qus: What are the three componets of Web?

  2. Qus: Define version control and give one example.

    Ans: Version control is known as a system that records changes to one or more files over time. This takes place so people can recall specific versions at another time. An example of version control is GitHub.

  4. Qus: Identify the odd one from these tags: span, p, a, i, and state the reason?

    Ans: The odd one is the i tag, because it italicizes text while span, p, and a deal with the structure of the document and are featured in the first HTML tags.

  5. Qus: What are the tags required to create a navigation list in HTML?

    Ans: The tags required to create a navigation list in HTML is the nav tag as well as the li tag. You can choose if you want the list to be ordered, ol, or unordered, ul.

  6. Qus: What element is used to create images? Name its attributes.

    Ans: The element used to create images is img. The attributes are src to specify the location, .jpg to represent the extension of the file, alt to specify alternative text, and width and height to specify width and height. There are also semantic tags such as figure to display the image and figcaption if there is a caption in the figure

  7. Qus: Identify the odd one from these tags: table, tr, b, td and state the reason

    Ans: The odd one from these tags is the b tag, because the other three tags are associated with making a table while the b tag is just boldening text.
